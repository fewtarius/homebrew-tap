# fewtarius-tap

Homebrew tap for software by fewtarius.

## Install

```
brew tap fewtarius/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| `img2ans` | Convert images to CP437 ANSI art for telnet BBSs and terminals |

### Install img2ans

```
brew install fewtarius/tap/img2ans
```

## Casks

| Cask | Description |
|------|-------------|
| `photonterm` | BBS terminal client - ANSI/VT100, SSH, Telnet, file transfer |

### Install PhotonTERM

```
brew install --cask fewtarius/tap/photonterm
```
